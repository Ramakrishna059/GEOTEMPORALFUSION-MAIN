# GeoTemporalFusion - AI Wildfire Prediction System

AI-powered wildfire risk prediction using geospatial and temporal data fusion.

## Features
- Real-time fire risk prediction
- Geospatial image analysis
- Temporal weather pattern integration
- FastAPI backend with REST API

## API Endpoints
- `GET /` - API documentation
- `GET /health` - Health check
- `POST /predict` - Fire risk prediction

## Usage
This Space runs a FastAPI server. Visit the API documentation at the root URL.
